This is a set of python scripts used to import the latest CSV files from the GIT
https://github.com/sapics/ip-location-db

It needs some configuration in order to get it working.

You will need to setup a python environment and mysql database to import the csv files

Once you have installed all pip related packages you can run the bash script to execute all the python imports

Create a directory to write the log files in a folder called "logs"
